# godfish on tap

[godfish](https://github.com/rafaelespinoza/godfish) is a database migration tool.
This repo is a [Homebrew tap](https://docs.brew.sh/Taps) for installing it.
Each release contains a separate binary per DB driver. At the time of this
writing, these DBs are supported:

* cassandra
* mysql
* postgres
* sqlite3
* sqlserver

There is a formula for each one, as well as a formula to install all of them.

## How do I install these formulae?

`brew install rafaelespinoza/godfish/<formula>`

Or `brew tap rafaelespinoza/godfish` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rafaelespinoza/godfish"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).

## What's in this repo?

Formula files are generated by fetching release data from the source repo and
putting that data into the ruby files.

* `Formula/`: Where Homebrew formula files are written to. Pretty standard.
* `generator`:
  * `generator/internal/`: Implements the formula generator, written in Go.
  * `generator/templates/`: Template files for generating ruby formula files.
  * `generator/main.go`: Entrypoint to formula generator.
