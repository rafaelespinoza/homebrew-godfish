# Rafaelespinoza Godfish

## How do I install these formulae?

`brew install rafaelespinoza/godfish/<formula>`

Or `brew tap rafaelespinoza/godfish` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rafaelespinoza/godfish"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
